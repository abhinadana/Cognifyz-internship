# Cognifyz Technologies — Web Development Internship
## Level 1 — Task 1

A simple, professional, and fully responsive webpage built with **only HTML5 and CSS3** (no Bootstrap, no JavaScript).

### Project Structure
```
Cognifyz-Level1-Task1/
│── index.html
│── style.css
│── images/
│── README.md
```

> Note: The hero image is loaded from Unsplash via a direct URL, so no local image
> file is required. If you'd prefer a local copy, download any web-development
> related image into the `images/` folder and update the `src` attribute in
> `index.html` accordingly.

### Features
- Semantic HTML5 structure (`header`, `nav`, `main`, `section`, `article`, `footer`)
- Responsive navigation bar with Home, About, and Contact links
- Hero section with heading, paragraph, call-to-action, and image
- About section with feature cards
- Contact form with label, placeholder text, and HTML5 validation
- Modern styling: CSS variables, Flexbox, gradients, rounded corners, shadows, and hover transitions
- Fully responsive layout for desktop, laptop, tablet, and mobile (via media queries)
- Accessible markup: alt text, labels, and good color contrast

### How to Run
1. Download or clone the `Cognifyz-Level1-Task1` folder.
2. Open `index.html` in any modern web browser (Chrome, Firefox, Edge).
3. No build tools, servers, or installations are required.
